# Команда `head`

`head` выводит **первые строки** файла.

Утилита обычно находится в `/usr/bin/head`.

Удобна для быстрого просмотра начала большого файла: заголовков, первых записей лога, структуры конфига.

## Базовое использование

Первые 10 строк файла (по умолчанию):

```bash
head /etc/passwd
```

Несколько файлов — для каждого будет заголовок с именем:

```bash
head file1 file2
```

Синтаксис:

```text
head [опции] [файл...]
```

Без файла `head` читает **stdin**:

```bash
ls -l | head
```

## Полезные опции

| Опция | Описание |
| ----- | -------- |
| `-n N` | Показать первые N строк |
| `-n -N` | Показать все строки **кроме** последних N |
| `-c N` | Показать первые N **байт** |
| `-q` | Не печатать имена файлов при нескольких аргументах |
| `-v` | Всегда печатать имена файлов |

Короткая запись `-n`:

```bash
head -5 file.txt    # то же, что head -n 5 file.txt
```

## Примеры

Первые 5 строк:

```bash
head -n 5 /var/log/syslog
head -5 /var/log/syslog
```

Все строки, кроме последних 3:

```bash
head -n -3 file.txt
```

Первые 20 байт:

```bash
head -c 20 file.bin
```

Посмотреть начало нескольких логов:

```bash
head -n 3 app.log app.old.log
```

Ограничить вывод длинной команды:

```bash
ps aux | head -10
history | head
```

## `head` и `tail`

| Команда | Что показывает |
| ------- | -------------- |
| `head` | **начало** файла |
| `tail` | **конец** файла |

```bash
head -20 file.txt
tail -20 file.txt
```

Для постраничного просмотра всего файла используют [less](less.md) или [more](more.md).
