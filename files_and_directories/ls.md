# Команда `ls`

`ls` (list) показывает содержимое каталога: файлы и папки.

Команда встроена в оболочку Bash как внешняя утилита и обычно находится в `/bin/ls`.

## Базовое использование

Показать файлы в текущей папке:

```bash
ls
```

Показать содержимое конкретного каталога:

```bash
ls /etc
ls ~/Documents
```

Несколько каталогов за один раз:

```bash
ls /bin /usr/bin
```

## Полезные опции


| Опция | Описание                                           |
| ----- | -------------------------------------------------- |
| `-l`  | Подробный список: права, владелец, размер, дата    |
| `-a`  | Показать скрытые файлы (начинаются с `.`)          |
| `-h`  | Размеры в удобном виде: `K`, `M`, `G` (с `-l`)     |
| `-t`  | Сортировка по времени изменения (новые сверху)     |
| `-r`  | Обратный порядок сортировки                        |
| `-R`  | Рекурсивно: содержимое подкаталогов                |
| `-1`  | По одному имени на строку                          |
| `-d`  | Информация о самом каталоге, а не о его содержимом |


Опции можно комбинировать:

```bash
ls -la
ls -lah
ls -ltr
```

## Примеры

Подробный список с размерами:

```bash
ls -lh
```

Все файлы, включая скрытые:

```bash
ls -a
```

Подробный список всех файлов:

```bash
ls -lah
```

Сортировка по времени (старые внизу):

```bash
ls -ltr
```

Дерево каталогов (рекурсивно):

```bash
ls -R
```

Показать сам каталог, а не его содержимое:

```bash
ls -ld /etc
```

## Что означает вывод `ls -l`

Пример строки:

```text
drwxr-xr-x  2 root root 4096 Jun 15 03:40 dir
-rw-r--r--  1 root root    0 Jun 15 03:40 file
```


| Часть          | Значение                       |
| -------------- | ------------------------------ |
| `d` / `-`      | Тип: `d` — каталог, `-` — файл |
| `rwxr-xr-x`    | Права доступа                  |
| `2`            | Число жёстких ссылок           |
| `root`         | Владелец                       |
| `root`         | Группа                         |
| `4096`         | Размер в байтах                |
| `Jun 15 03:40` | Дата и время изменения         |
| `dir`          | Имя файла или каталога         |


## Цветной вывод

Во многих системах `ls` подсвечивает типы файлов цветом.
Проверить, включена ли раскраска:

```bash
ls --color=auto
```

Отключить цвета:

```bash
ls --color=none
```

## Алиасы

В интерактивной оболочке `ls` часто заменён алиасом:

```bash
alias ls
```

Типичный пример:

```text
alias ls='ls --color=auto'
```

Чтобы вызвать команду без алиаса:

```bash
\ls
command ls
```
