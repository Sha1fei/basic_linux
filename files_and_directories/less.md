# Команда `less`

`less` постранично показывает текст и позволяет **листать вперёд и назад** — основной просмотрщик файлов в терминале.

Утилита обычно находится в `/usr/bin/less`.

Название — игра слов: «less is more» (меньше — это больше), в противовес [more](more.md).

## Базовое использование

Открыть файл:

```bash
less /var/log/syslog
```

Просмотр вывода команды:

```bash
ls -l /usr/bin | less
grep -r TODO project/ | less
```

Синтаксис:

```text
less [опции] [файл...]
```

## Управление внутри `less`

### Навигация

| Клавиша | Действие |
| ------- | -------- |
| `Пробел` / `f` | Страница вперёд |
| `b` | Страница назад |
| `Enter` / `j` | Одна строка вперёд |
| `k` | Одна строка назад |
| `g` | В начало файла |
| `G` | В конец файла |
| `q` | Выход |

### Поиск

| Клавиша | Действие |
| ------- | -------- |
| `/текст` | Искать вперёд |
| `?текст` | Искать назад |
| `n` | Следующее совпадение |
| `N` | Предыдущее совпадение |

### Прочее

| Клавиша | Действие |
| ------- | -------- |
| `h` | Справка |
| `F` | Режим follow (как `tail -f`), `Ctrl+C` — вернуться |
| `&pattern` | Показать только строки, совпадающие с pattern |

## Полезные опции

| Опция | Описание |
| ----- | -------- |
| `-N` | Нумерация строк |
| `-S` | Длинные строки обрезать, не переносить |
| `-i` | Поиск без учёта регистра |
| `-F` | Выйти сразу, если файл помещается на экран (как `cat`) |
| `-R` | Показывать ANSI-цвета (удобно с `grep --color`) |
| `+N` | Открыть с строки N |
| `+/шаблон` | Открыть с первого совпадения |

```bash
less -N file.txt
less -R grep_output.txt
less +100 file.txt
less +/error app.log
grep --color=always error app.log | less -R
```

## Примеры

Просмотр конфига:

```bash
less /etc/nginx/nginx.conf
```

С номерами строк:

```bash
less -N script.sh
```

Большой лог с поиском:

```bash
less /var/log/syslog
# внутри: /error  затем n для следующих
```

Следить за растущим файлом:

```bash
less +F /var/log/app.log
# Ctrl+C — выйти из follow, q — закрыть less
```

## `less`, `more`, `head`, `tail`

| Задача | Команда |
| ------ | ------- |
| Первые N строк | [head](head.md) |
| Последние N строк | [tail](tail.md) |
| Весь файл постранично | `less` |
| Простой постраничный просмотр | [more](more.md) |
| Свежие строки лога | `tail -f` или `less +F` |

```bash
head -20 file.txt    # быстро начало
tail -20 file.txt    # быстро конец
less file.txt        # весь файл интерактивно
```

## Переменная `PAGER`

Многие команды (`man`, `git log`) используют постраничный просмотр через переменную окружения:

```bash
echo $PAGER
export PAGER=less
man ls
```
