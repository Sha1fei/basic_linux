# Команда `man`

`man` (manual) открывает **справочную страницу** (man-страницу) по команде или теме.

Утилита обычно находится в `/usr/bin/man`.

## Базовое использование

Справка по команде:

```bash
man ls
man cp
man mkdir
```

Выйти из просмотра:

```text
q
```

## Разделы man

Документация разбита на разделы. Чаще всего нужен раздел **1** — пользовательские команды.

| Раздел | Содержимое |
| ------ | ---------- |
| 1 | Команды shell и утилиты |
| 5 | Форматы файлов (`passwd`, `hosts`) |
| 8 | Администрирование (часто требует root) |

Указать раздел явно:

```bash
man 1 ls
man 5 passwd
```

## Полезные опции

| Опция | Описание |
| ----- | -------- |
| `-k` | Поиск по ключевым словам (как `apropos`) |
| `-f` | Краткое описание команды (как `whatis`) |
| `-a` | Показать все подходящие man-страницы |
| `-w` | Только путь к man-странице, без открытия |

```bash
man -k copy
man -f ls
man -w cp
```

## Навигация внутри `man`

| Клавиша | Действие |
| ------- | -------- |
| `Пробел` / `Page Down` | следующая страница |
| `b` / `Page Up` | предыдущая страница |
| `/текст` | поиск вперёд |
| `n` | следующее совпадение |
| `h` | помощь по навигации |
| `q` | выход |

Пример поиска внутри страницы:

```text
/copy
```

## Примеры

Открыть справку по `rm`:

```bash
man rm
```

Найти команды по теме:

```bash
man -k directory
apropos directory
```

Краткое описание:

```bash
man -f who
whatis who
```

Узнать путь к man-странице:

```bash
man -w ls
# /usr/share/man/man1/ls.1.gz
```
