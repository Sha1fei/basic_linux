# Команда `touch`

`touch` создаёт пустые файлы или обновляет время доступа и изменения существующих файлов.

Утилита обычно находится в `/usr/bin/touch`.

## Базовое использование

Создать пустой файл:

```bash
touch file
```

Если файл уже существует, `touch` не меняет его содержимое — только метки времени.

Создать несколько файлов за один раз:

```bash
touch file1 file2 file3
```

Создать файл по указанному пути:

```bash
touch ~/dir/notes.txt
```

## Полезные опции

| Опция | Описание |
| ----- | -------- |
| `-a` | Обновить только время доступа (access time) |
| `-m` | Обновить только время изменения (modification time) |
| `-c` | Не создавать файл, если его нет |
| `-t` | Задать время в формате `[[CC]YY]MMDDhhmm[.ss]` |
| `-d` | Задать дату строкой, например `"2024-01-15 12:00"` |

Опции можно комбинировать:

```bash
touch -am file
touch -c file
```

## Примеры

Создать файл и проверить результат:

```bash
touch file
ls -l file
```

Создать несколько файлов:

```bash
touch file1 file2 file3
ls -l
```

Обновить время изменения существующего файла:

```bash
touch -m file
```

Не создавать файл, если его нет:

```bash
touch -c missing.txt
```

Задать конкретную дату:

```bash
touch -d "2024-06-15 10:30" file
```

Задать время в формате `-t`:

```bash
touch -t 202406151030 file
```

## Метки времени файла

У каждого файла есть два основных времени:

| Метка | Описание |
| ----- | -------- |
| **atime** (access time) | Когда файл последний раз читали |
| **mtime** (modification time) | Когда содержимое файла последний раз меняли |

Посмотреть метки времени:

```bash
ls -l file
stat file
```
