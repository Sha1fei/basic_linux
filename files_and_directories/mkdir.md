# Команда `mkdir`

`mkdir` (make directory) создаёт новые каталоги.

Утилита обычно находится в `/bin/mkdir` или `/usr/bin/mkdir`.

## Базовое использование

Создать один каталог:

```bash
mkdir dir
```

Создать несколько каталогов за один раз:

```bash
mkdir dir1 dir2 dir3
```

Создать каталог по указанному пути:

```bash
mkdir ~/projects/new_dir
```

Проверить результат:

```bash
ls -ld dir
```

## Полезные опции

| Опция | Описание |
| ----- | -------- |
| `-p` | Создать всю цепочку каталогов; не ругаться, если каталог уже есть |
| `-m` | Задать права при создании, например `-m 755` |
| `-v` | Показать, какие каталоги были созданы |

Опции можно комбинировать:

```bash
mkdir -pv dir/subdir/nested
```

## Примеры

Создать каталог в текущей папке:

```bash
mkdir dir
ls -l
```

Создать вложенные каталоги одной командой:

```bash
mkdir dir/subdir/nested
# mkdir: cannot create directory 'dir/subdir/nested': No such file or directory

mkdir -p dir/subdir/nested
ls -R dir
```

Создать каталог с правами:

```bash
mkdir -m 755 mydir
ls -ld mydir
```

Создать несколько уровней и увидеть процесс:

```bash
mkdir -pv a/b/c
```

## Опция `-p` — самая важная

Без `-p` `mkdir` создаёт только **один** уровень. Родительский каталог должен уже существовать.

С `-p` команда:

- создаёт все недостающие каталоги в пути;
- не выдаёт ошибку, если конечный каталог уже существует.

```bash
mkdir -p dir/subdir
mkdir -p dir/subdir   # ошибки не будет
```
