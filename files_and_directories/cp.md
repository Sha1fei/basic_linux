# Команда `cp`

`cp` (copy) копирует файлы и каталоги.

Утилита обычно находится в `/bin/cp` или `/usr/bin/cp`.

## Базовое использование

Скопировать файл:

```bash
cp source.txt copy.txt
```

Скопировать файл в каталог:

```bash
cp file.txt dir/
```

Скопировать несколько файлов в каталог:

```bash
cp file1 file2 file3 dir/
```

Синтаксис:

```text
cp [опции] источник назначение
cp [опции] файл1 файл2 ... каталог
```

## Полезные опции

| Опция | Описание |
| ----- | -------- |
| `-r` | Рекурсивно: копировать каталоги с содержимым |
| `-i` | Спрашивать перед перезаписью |
| `-f` | Перезаписывать без вопросов |
| `-v` | Показать, что копируется |
| `-p` | Сохранить права, владельца и метки времени |
| `-a` | Архивное копирование: `-rp` + сохранить ссылки и атрибуты |
| `-u` | Копировать только если источник новее или файл ещё не существует |

Опции можно комбинировать:

```bash
cp -rv dir dir_backup
cp -av project project_copy
```

## Примеры

Скопировать файл:

```bash
touch file
cp file file_copy
ls -l file file_copy
```

Скопировать в другой каталог:

```bash
mkdir backup
cp file backup/
ls backup/
```

Скопировать каталог:

```bash
mkdir -p dir/subdir
touch dir/file dir/subdir/a
cp -r dir dir_copy
ls -R dir_copy
```

Спросить перед перезаписью:

```bash
cp -i file.txt backup/
```

Сохранить права и время:

```bash
cp -p file.txt copy.txt
```

Полная копия каталога:

```bash
cp -a dir dir_backup
```
