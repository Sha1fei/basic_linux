# Команда `id`

`id` показывает **идентификаторы** текущего пользователя: uid, gid и список групп.

Утилита обычно находится в `/usr/bin/id`.

## Базовое использование

Информация о текущем пользователе:

```bash
id
```

Пример вывода:

```text
uid=1000(alex) gid=1000(alex) groups=1000(alex),27(sudo)
```

Что означает:

| Часть | Значение |
| ----- | -------- |
| `uid` | числовой ID пользователя |
| `gid` | основная группа |
| `groups` | все группы, в которых состоит пользователь |

## Полезные опции

| Опция | Описание |
| ----- | -------- |
| `-u` | Только uid |
| `-g` | Только gid |
| `-G` | Только дополнительные группы |
| `-n` | Показать имена вместе с числами (с `-u`, `-g`, `-G`) |
| `-r` | Реальный uid/gid (редко нужно) |

```bash
id -u
id -gn
id -Gn
```

## Примеры

Текущий пользователь:

```bash
id
```

Только uid:

```bash
id -u
# 0
```

Имя основной группы:

```bash
id -gn
# root
```

Информация о другом пользователе:

```bash
id alex
```

## uid и root

Пользователь `root` обычно имеет `uid=0`:

```bash
id
# uid=0(root) gid=0(root) groups=0(root)
```

В Linux `uid 0` — суперпользователь с полными правами.

## `id` и `whoami`

```bash
whoami
# alex

id -un
# alex
```

`whoami` проще, `id` — подробнее.
