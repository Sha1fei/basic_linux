# Файл `/etc/group`

`/etc/group` — системный файл со **списком групп** Linux и их участников.

Обычный пользователь может **читать** этот файл.

## Просмотр

Показать весь файл:

```bash
cat /etc/group
```

Первые строки:

```bash
head /etc/group
```

Найти конкретную группу:

```bash
grep sudo /etc/group
```

## Формат строки

Пример:

```text
sudo:x:27:alex
```

| Поле | Пример | Значение |
| ---- | ------ | -------- |
| 1 | `sudo` | имя группы |
| 2 | `x` | пароль группы: `x` значит «смотри в `/etc/gshadow`» |
| 3 | `27` | gid — ID группы |
| 4 | `alex` | список пользователей в группе (через запятую) |

Разделитель полей — двоеточие `:`.

Если в 4-м поле никого нет, группа существует, но без дополнительных участников:

```text
nogroup:x:65534:
```

## Примеры

Посмотреть группу `root`:

```bash
grep ^root: /etc/group
# root:x:0:
```

Группы текущего пользователя через команду:

```bash
groups
id
```

## `/etc/group` и `/etc/gshadow`

| Файл | Содержимое | Кто может читать |
| ---- | ---------- | ---------------- |
| `/etc/group` | имена групп, gid, участники | все |
| `/etc/gshadow` | пароли групп | обычно только root |

Пароли групп используются редко. Чаще доступ даётся через членство в группе.

```bash
cat /etc/gshadow
# Permission denied   (если вы не root)
```

## Команда `usermod`

`usermod` изменяет учётную запись пользователя. Для работы с группами нужны права **root**.

Утилита обычно находится в `/usr/sbin/usermod`.

### Добавить пользователя в группу

Добавить `alex` в группу `sudo`, **не удаляя** другие группы:

```bash
sudo usermod -aG sudo alex
```

**Важно:** с опцией `-G` всегда используйте `-a` (append). Иначе старые дополнительные группы будут **заменены**:

```bash
sudo usermod -G sudo alex   # опасно: останется только sudo
sudo usermod -aG sudo alex  # правильно: sudo добавится к существующим
```

Проверить результат:

```bash
id alex
grep sudo /etc/group
```

Изменения вступают в силу при **следующем входе** пользователя. В уже открытой сессии:

```bash
newgrp sudo
```

### Другие полезные опции

| Опция | Описание |
| ----- | -------- |
| `-aG группа` | Добавить в дополнительную группу |
| `-G группы` | Заменить все дополнительные группы |
| `-g группа` | Сменить **основную** группу |
| `-l новое_имя` | Переименовать пользователя |
| `-d каталог` | Сменить домашний каталог |
| `-s shell` | Сменить shell |
| `-L` | Заблокировать учётную запись |
| `-U` | Разблокировать учётную запись |

Примеры:

```bash
sudo usermod -g developers alex
sudo usermod -d /home/alex_new -m alex
sudo usermod -s /bin/bash alex
```
