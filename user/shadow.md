# Файл `/etc/shadow`

`/etc/shadow` — системный файл с **хэшами паролей** и данными об их сроке действия.

Читать его может обычно **только root**. Обычный пользователь получит `Permission denied`.

## Просмотр

От имени root:

```bash
sudo cat /etc/shadow
```

или, если вы уже root:

```bash
cat /etc/shadow
```

Найти конкретного пользователя:

```bash
sudo grep ^alex: /etc/shadow
```

## Формат строки

Пример:

```text
root:$6$rounds=...$hash...:19800:0:99999:7:::
```

| Поле | Пример | Значение |
| ---- | ------ | -------- |
| 1 | `root` | имя пользователя |
| 2 | `$6$...` | хэш пароля (не сам пароль) |
| 3 | `19800` | дней с 01.01.1970 до последней смены пароля |
| 4 | `0` | минимальный срок до следующей смены |
| 5 | `99999` | максимальный срок действия пароля |
| 6 | `7` | за сколько дней предупредить о смене |
| 7 | пусто | дней до блокировки после истечения срока |
| 8 | пусто | дата блокировки учётной записи |
| 9 | пусто | зарезервировано |

Разделитель полей — двоеточие `:`.

## Особые значения во 2-м поле

| Значение | Значение |
| -------- | -------- |
| `*` | вход запрещён |
| `!` | пароль заблокирован |
| `$1$`, `$5$`, `$6$` | хэш пароля (разные алгоритмы) |

Пароль в открытом виде **никогда** не хранится — только хэш.

## Примеры

Попытка прочитать без прав:

```bash
cat /etc/shadow
# cat: /etc/shadow: Permission denied
```

Просмотр от root:

```bash
sudo head -3 /etc/shadow
```

Проверить, есть ли пароль у пользователя:

```bash
sudo grep ^alex: /etc/shadow
```

## `/etc/passwd` и `/etc/shadow`

| Файл | Содержимое | Кто может читать |
| ---- | ---------- | ---------------- |
| `/etc/passwd` | имена, uid, домашние каталоги | все |
| `/etc/shadow` | хэши паролей, сроки действия | обычно только root |

В `/etc/passwd` на месте пароля стоит `x`:

```text
alex:x:1000:1000:...
```

Это значит: «пароль смотри в `/etc/shadow`».

## Безопасность

- не копируйте и не публикуйте `/etc/shadow`
- не редактируйте файл вручную без понимания формата
- для смены пароля используйте `passwd`, а не прямое редактирование
