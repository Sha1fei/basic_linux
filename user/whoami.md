# Команда `whoami`

`whoami` показывает **имя текущего пользователя** — под кем вы сейчас работаете в shell.

Утилита обычно находится в `/usr/bin/whoami`.

## Базовое использование

```bash
whoami
```

Пример вывода:

```text
root
```

или

```text
alex
```

## Примеры

Узнать текущего пользователя:

```bash
whoami
```

Сравнить с переменной окружения:

```bash
whoami
echo $USER
```

Обычно результат совпадает.

Проверить перед командой, требующей прав:

```bash
whoami
rm /etc/hosts   # может не сработать, если вы не root
```

## `whoami` и `who`

| Команда | Что показывает |
| ------- | -------------- |
| `whoami` | только **ваше** имя |
| `who` | **всех**, кто вошёл в систему |

```bash
whoami
# root

who
# root     pts/0   ...
```

## `whoami` и `id`

`whoami` показывает только имя. `id` — подробности: uid, gid, группы.

```bash
whoami
# alex

id
# uid=1000(alex) gid=1000(alex) groups=1000(alex),27(sudo)
```

## Связанные команды

- `id` — uid, gid и группы
- `who` — все активные входы
- `w` — кто вошёл и чем занят
